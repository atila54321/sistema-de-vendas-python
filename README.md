# Sistema de Vendas em Python

Projeto desenvolvido em Python para gerenciamento de produtos e vendas. O sistema permite cadastrar produtos, realizar vendas, controlar estoque e visualizar estatísticas.

## Funcionalidades

- Cadastro de produtos
- Listagem de produtos
- Busca de produtos
- Realização de vendas
- Histórico de vendas
- Controle de estoque
- Reposição de produtos
- Remoção de produtos
- Estatísticas de vendas
- Validação de entradas

## Tecnologias Utilizadas

- Python 3

## Estrutura dos Dados

Cada produto é armazenado em um dicionário contendo:

- Nome
- Preço
- Quantidade em estoque

As vendas são registradas em um histórico contendo:

- Produto vendido
- Quantidade
- Valor total

## Objetivo

Projeto criado para praticar:

- Lógica de programação
- Estruturas condicionais
- Laços de repetição
- Listas e dicionários
- Manipulação de dados
- Validação de entradas

## Como executar

```bash
python main.py
```
